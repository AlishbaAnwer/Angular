# Beginner Angular Demonstration

## About
Demo to describe the following:
* What is a MVC
* What is a Controller?
* What is a Directive?
* ng-model
* ng-controller
* ng-repeat
* ng-show & ng-hide
* ng-click
* ng-class
* filters

## Technologies
* Angular
* Stylus
* Grunt
* NPM

## Getting Started
1. Install grunt dependencies
	
	```
	npm install
	```
2. Update package.json
	* Update your Name, Description, Repository, Homepage etc.
3. Run Grunt
	
	``` 
	grunt
	```
